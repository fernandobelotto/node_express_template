## About The Project

This is an simple base project for creating rapid REST Api using NodeJS and Express.

### Built With

| Lib         | Version |
| ----------- | ------- |
| express     | 4.17.1  |
| body-parser | 1.19.0  |
| cors        | 2.8.5   |
| dotenv      | 8.2.0   |
| mysql       | 2.18.1  |
| mysql2      | 2.2.5   |
| pg          | 8.5.1   |
| pg-hstore   | 2.3.3   |
| sequelize   | 6.5     |
